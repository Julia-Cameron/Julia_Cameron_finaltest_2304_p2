## 📝 Description
JSEK Marketplace is a web-based platform designed to buy and sell new and used electronics. This repository contains the front-end architecture and UI mockups for the core user experience.

## 🚀 Features
- **Responsive Landing Page:** Engaging UI to showcase marketplace items.
- **Authentication Flow:** Dedicated Sign-up and Login interfaces.
- **Seamless Checkout:** A streamlined mockup for finalizing transactions.

## 🛠️ Tech Stack
- **HTML5:** Semantic structure for accessibility.
- **CSS3:** Custom styling featuring [e.g., Flexbox/Grid] and responsive breakpoints.
- **JavaScript (ES6+):** Logic & Interactivity: Handles form validation for the Signup and Login pages.
  - **DOM Manipulation:** Dynamically updates UI elements based on user interaction (e.g., updating the cart total).
- **Web Storage API (LocalStorage):** Used to persist user data across sessions, such as maintaining a "logged-in" state or saving items added to the cart in the checkout.html flow.

## 📂 Project Structure
- `index.html` - Homepage
- `style.css` - Global styles
- `signup.html` / `login.html` - User Auth pages
- `checkout.html` - Payment/Finalization mockup

## 🚦 How to Run
1. Clone the repository.
2. Open `index.html` in any modern web browser.
